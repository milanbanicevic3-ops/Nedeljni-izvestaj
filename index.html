Adding a new report - 3 steps 1. Upload the HTML file named YYYY-MM-DD.html to the repository.
2. Add a new entry at the top of reports.json:
{
  "date": "2026-04-24",
  "title": "Weekly Synthesis. Description here.",
  "thesis": "One-paragraph summary.",
  "bias": { "EURUSD": "bullish", "XAUUSD": "bearish" },
  "highlights": ["Key stat 1", "Key stat 2"],
  "file": "2026-04-24.html"
}
3. Commit. Wait 2 minutes. Done.

JSON rule: ASCII only. No middle dots, em dashes, or Unicode. Periods and hyphens only in string values.
