Adding a New Report
Follow these 3 steps — takes under 30 seconds once you have the HTML file ready
1
Upload the HTML report file to GitHub
In the repo, click Add file → Upload files. Name the file with the date format:
YYYY-MM-DD.html e.g. 2026-03-29.html
Commit the file.
2
Open reports.json and add a new entry
Click reports.json in the repo, then the pencil (edit) icon. Add your new report at the top of the array (newest first):
{ "date": "2026-03-29", "title": "Short headline · Key number", "thesis": "One sentence macro summary.", "bias": { "XAUUSD": "bullish", "EURUSD": "bearish", "USDJPY": "neutral" }, "highlights": ["Oil $112", "DXY 101.2"], "file": "2026-03-29.html" },
3
Commit reports.json — done
Scroll down, click Commit changes. GitHub Pages updates in ~60 seconds. The new report will appear at the top of the archive automatically. Your team refreshes the page and it's there.
Bias values accepted
bullish bearish neutral
